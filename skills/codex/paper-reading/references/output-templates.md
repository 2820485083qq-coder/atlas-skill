# Output Templates

Use these templates when the user asks for a specific reading style or when a blank-page problem would slow you down.

## Quick Summary

```markdown
## 一句话
[用一句话概括论文在解决什么问题，以及核心办法是什么。]

## 核心内容
- 研究问题：[一句话]
- 核心方法：[2-3 句]
- 关键证据：[1-2 条最重要结果]
- 主要局限：[1 条]

## 是否值得继续读
[说明适合谁继续精读，以及理由。]
```
