# 06-source-index

This source index is intentionally short and points to the highest-value anchors used in the skill.

## Core professional references

1. SNAME, Principles of Naval Architecture
   - https://sname.org/principles-naval-architecture
   - Use for stability, strength, resistance, propulsion, flow, controllability, and fundamentals.

2. SNAME, Ship Structural Analysis and Design, Owen Hughes and Jeom Kee Paik
   - https://sname.org/ship-structural-analysis-and-design
   - Use for rational preliminary structural design and optimization.

3. J. N. Newman, Marine Hydrodynamics, MIT Press
   - https://mitpress.mit.edu/9780262534826/marine-hydrodynamics/
   - Use for hydrodynamic fundamentals and theory/empirics balance.

4. Rawson and Tupper, Basic Ship Theory, ScienceDirect
   - https://www.sciencedirect.com/book/monograph/9780750653985/basic-ship-theory
   - Use for foundational geometry, flotation, stability, dynamics, seakeeping, and design.

## Seminal methods and papers

5. Holtrop and Mennen, "An approximate power prediction method"
   - https://journals.sagepub.com/doi/10.3233/ISP-1982-2933501
   - Use for first-pass resistance/power prediction with domain checks.

6. Daniel Savitsky, "Hydrodynamic Design of Planing Hulls"
   - https://cir.nii.ac.jp/crid/1360861295038811520
   - Use for planing hull dynamic equilibrium and porpoising constraints.

7. Gerritsma, Onnink, Versluis, Delft systematic yacht hull series
   - https://journals.sagepub.com/doi/10.3233/ISP-1981-2832801
   - Use for systematic hull-form series and sailing-yacht prediction logic.

8. Michell thin-ship theory derivative reference
   - https://www.jstage.jst.go.jp/article/jjasnaoe/19/0/19_19/_article
   - Use as pointer to wave-resistance theory; verify against original paper if doing scholarly work.

## Procedures and rules

9. ITTC Recommended Procedure 7.5-02-02-01, Resistance Test
   - https://ittc.info/media/9595/75-02-02-01.pdf
   - Use for model-test procedure and extrapolation quality.

10. IMO International Code on Intact Stability 2008
   - https://imo-epublications.org/content/books/9789280117202
   - Use for intact stability criteria and documentation.

11. IACS Common Structural Rules
   - https://iacs.org.uk/resolutions/common-structural-rules
   - Use for bulk carrier and tanker structural standards.

12. IMO GHG emissions work and 2023 GHG Strategy
   - https://www.imo.org/en/mediacentre/hottopics/pages/cutting-ghg-emissions.aspx
   - Use for carbon and energy-efficiency constraints.
