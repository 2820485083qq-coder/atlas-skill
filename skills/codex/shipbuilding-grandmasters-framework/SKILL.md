# 船舶工程宗师评判框架

## 使用定位

这是一个“主题框架型 Skill”，不是某个大师的角色扮演。它把船舶与海洋工程中的经典大师、权威组织和现代规范思路蒸馏成一套可执行的评判体系。

适用场景：
- 评审船舶/海工/水动力/结构/稳性/推进方案是否靠谱。
- 帮用户理解船舶工程论文、毕业设计、答辩稿、仿真方案。
- 审查 CFD、模型试验、经验公式、规范计算是否适用。
- 对船型优化、节能装置、绿色船舶、改装方案做风险判断。
- 给出“下一步该补什么证据”的工程化建议。

不适用场景：
- 不替代船级社审图、注册验船师意见、法定检验或安全认证。
- 不在缺少尺寸、载荷、工况、规范版本时给确定结论。
- 不把单一公式、单一 CFD 图或单一经验判断当成工程真相。

## 核心原则

船不是一个水动力外形，也不是一堆规范条文。船是一个在海况、载荷、规范、建造、运营、经济和碳约束中工作的耦合系统。

评判任何船舶工程方案时，先问：

1. 它的任务和工况定义清楚了吗？
2. 它的物理机制闭合吗？
3. 它的方法适用于这个速度、船型、尺度和载荷吗？
4. 它的证据能从理论、试验、规范或验证过的数值方法追溯吗？
5. 它能通过结构、稳性、规范、建造和运营的共同约束吗？

## 回答工作流

### Step 1: 识别问题类型

先把用户问题归类：

| 类型 | 典型输入 | 处理方式 |
|---|---|---|
| 方案评审 | “这个船型方案靠谱吗？”“帮我审一下设计” | 使用 8 门评判体系，指出红旗和补证据路径 |
| 论文/答辩理解 | “帮我理解这篇论文”“老师会问什么” | 把论文内容映射到物理机制、方法、证据和局限 |
| 水动力/CFD | “网格怎么划”“阻力怎么算”“VOF 对吗” | 先查流动机制、尺度相似、方法适域和验证 |
| 结构/稳性/规范 | “强度够吗”“GM 合格吗”“规范怎么说” | 先明确规则版本、船型、载荷、工况和失效模式 |
| 绿色船舶/能效 | “EEXI/CII 怎么看”“氨燃料可行吗” | 同时看碳指标、燃料体积、安全、运营剖面和改装空间 |

如果涉及现行规范、法规、船级社要求、IMO 指标、EEXI/CII、材料标准或当前行业政策，必须先查询最新权威来源。

### Step 2: 收集最少必要输入

如果用户未提供信息，先列出缺口；能合理假设时要明说是假设。

最低输入清单：
- 船型与任务：货船、油船、散货船、集装箱船、客船、军船、艇、海工平台等。
- 主尺度：L、B、T、D、排水量、Cb、Cp、LCB、吃水、航速。
- 工况：设计航速、海况、载况、航线、吃水范围、操纵/耐波要求。
- 方法：经验公式、模型试验、CFD、FEA、规范计算、实船试航。
- 证据：图表、网格、边界条件、试验规程、规范版本、误差/不确定度。

### Step 3: 通过 8 个检查门

对复杂问题，按下列顺序评判。不要只挑用户最关心的一门，因为船舶设计的风险常常出现在邻近门。

#### 门 1：任务与运营剖面

判断问题：
- 设计目标是载货量、航速、续航、稳性、甲板面积、隐身、经济性还是绿色指标？
- 运营剖面是长期低速、定线高速、复杂海况、港内操纵还是极端任务？
- 是否区分设计点、常用点和极端点？

红旗：
- 只有“提高效率”但没有航速、载重、航线和海况。
- 单点优化，没有运营剖面。

#### 门 2：浮性、稳性、装载与纵倾

宗师镜片：IMO IS Code + Rawson/Tupper。

判断问题：
- 排水量、浮心、重心、GM、GZ 曲线、自由液面和装载状态是否一致？
- 是否考虑完整稳性、破舱稳性、风浪影响、下沉/纵倾和下浸点？
- 稳性结论是否只靠一个正 GM？

红旗：
- “GM 为正，所以稳定。”
- 没有 KG 来源、没有自由液面修正、没有 GZ 曲线或载况组合。

#### 门 3：阻力、推进与尺度相似

宗师镜片：Froude + Taylor + Michell + Newman + Holtrop/Mennen + Savitsky。

判断问题：
- 阻力是否分解为摩擦、剩余/兴波、形状、附体、空气、粗糙度和波浪增阻？
- 使用的方法是否适用于船型和速度范围？
- 模型试验是否说明 Froude 相似、Reynolds 修正、相关裕度和 ITTC 程序？
- 推进是否考虑螺旋桨敞水效率、伴流、推力减额、船身效率和空化？

红旗：
- 只给总阻力曲线，不解释组成。
- Holtrop、Savitsky 或系列方法被用于明显不适用的船型。
- CFD 没有网格无关性、时间步长无关性、边界距离、自由液面或验证。

#### 门 4：耐波、操纵与动态载荷

宗师镜片：Faltinsen + SNAME PNA。

判断问题：
- 是否考虑真实海况下的升沉、纵摇、横摇、砰击、甲板上浪、加速度和波浪增阻？
- 操纵性是否包括回转、Z 形、航向稳定、浅水/受限水域影响？
- 动态载荷是否会反过来影响结构、设备、人员舒适性和运营限制？

红旗：
- 只讨论静水阻力，不讨论目标海况。
- 只讲稳性，不讲横摇周期、阻尼或参数横摇风险。

#### 门 5：结构、疲劳、极限强度与可建造性

宗师镜片：Hughes/Paik + PNA Strength + IACS CSR。

判断问题：
- 全船梁弯矩、局部板架、屈曲、疲劳、焊接细节、砰击/碰撞/搁浅是否分别检查？
- FEA 的边界条件、载荷、网格和约束是否有物理意义？
- 材料、焊接、装配、公差、涂层、防腐和维修可达性是否考虑？

红旗：
- “规范过了，所以结构一定安全。”
- FEA 彩图没有载荷路径、边界条件和失效模式解释。
- 减重方案没有疲劳、振动、焊接和维修后果。

#### 门 6：规范、船级、旗国与安全

宗师镜片：IMO + IACS + 船级社规则。

判断问题：
- 适用的 IMO 公约、船级规则、旗国要求、港口国要求是否明确？
- 规范版本是否最新？
- 是否区分强制要求、推荐要求和公司/船东附加要求？

红旗：
- 没有规范版本。
- 用通用说法替代具体条款。
- 用“行业惯例”绕过明确强制标准。

#### 门 7：生命周期、运营、维护与碳约束

宗师镜片：IMO GHG Strategy + 现代系统工程。

判断问题：
- 是否考虑 EEDI/EEXI/CII 或相关能效/碳强度要求？
- 替代燃料是否检查体积、重量、安全区、通风、消防、供应链和运营成本？
- 是否有改装空间、维护可达性和报废/回收考虑？

红旗：
- 绿色方案只看主机效率，不看燃料舱、载货损失和安全系统。
- 单纯降速满足指标，但不看商业航线和交付窗口。

#### 门 8：证据质量与不确定性

宗师镜片：ITTC + Newman 的理论/经验平衡。

判断问题：
- 结论来自理论推导、经验公式、模型试验、CFD、实船试航还是规范计算？
- 是否有验证、误差范围、敏感性分析和适用边界？
- 关键假设是否公开？

红旗：
- 没有不确定度。
- 只有图片，没有数据。
- 只有“软件算出来”，没有方法和验证。

## 评分方法

每个检查门给 0-3 分：

- 0：缺失或无证据。
- 1：提到但没有量化。
- 2：有量化，但方法适域、验证或假设不完整。
- 3：量化充分，方法适域，假设清楚，有规范/试验/验证/来源支撑。

硬性规则：
- 稳性、结构、规范、安全、证据质量任何一项为 0，都不能给“方案靠谱”的结论。
- 总分高但任务分低，说明技术漂亮但可能商业/任务错误。
- 规范分高但物理分低，说明可能“过最低线”但缺乏鲁棒性。

## 常用追问

当信息不足时，优先问这些：

1. 船型、主尺度、排水量、设计航速和载况是什么？
2. 你现在用的是经验公式、CFD、模型试验、FEA 还是规范计算？
3. 想证明的是阻力降低、功率降低、稳性满足、结构安全，还是绿色指标改善？
4. 有无网格无关性、时间步长无关性、模型试验或实船数据验证？
5. 适用规范/船级社/IMO 条款是哪一版？

## 方法适域速查

| 方法 | 合适用途 | 必查边界 |
|---|---|---|
| Holtrop/Mennen | 常规排水型船初步阻力/功率估算 | 船型、Fn、Cb、主尺度比例、附体、数据库适配 |
| Savitsky | 棱形滑行艇/高速艇初步性能 | 滑行状态、纵倾、升力、湿长、失稳/海豚运动 |
| 系列船型方法 | 同族船型快速设计 | 系列参数范围和几何相似性 |
| CFD | 局部流动、自由液面、尾流、复杂非定常 | 网格、边界、湍流模型、时间步长、验证 |
| 模型试验 | 阻力、推进、耐波、操纵关键证据 | ITTC/试验程序、尺度效应、误差、不确定度 |
| FEA | 结构局部/全船响应和热点分析 | 载荷、边界条件、网格、材料、失效模式 |
| 规范计算 | 合规证明 | 版本、船型适用性、附加符号、解释文件 |

## 诚实边界

- 该框架能帮助发现逻辑漏洞和证据缺口，但不能替代船级社审图、试验水池报告或法定检验。
- 对现行规则、环保指标、船级社要求必须查最新官方来源。
- 对新船型、新燃料、新材料、新结构，应提高一手证据要求；经验公式和老系列方法只能作为初筛。
- 如果用户不给主尺度、载况、航速、海况和方法来源，只能给条件性判断。

## 示例

用户：帮我看这个 CFD 阻力结果能不能写进论文。

回答重点：
- 先判断是否有计算域、自由液面、湍流模型、网格无关性、时间步长、残差/监测量、验证数据。
- 再判断是否分解摩擦阻力、压差阻力、兴波阻力或至少解释总阻力变化机制。
- 最后给出能否作为“趋势证据”或“定量结论”的分级。

用户：我这个船 GM 为正，是不是稳性没问题？

回答重点：
- 立即指出 GM 为正不是充分条件。
- 要求 GZ 曲线、装载工况、自由液面修正、下浸点、风浪/天气准则、完整/破舱适用规范。
